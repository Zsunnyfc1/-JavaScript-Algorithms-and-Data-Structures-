Palindrome Checker

A palindrome is a word or sentence that's spelled the same way both forward and backward, ignoring punctuation, case, and spacing.

Enter text into the box and see if it is a palindrome.

Roman Numeral Converter

Roman Numeral Symbols

  • I = 1
  • V = 5
  • X = 10
  • L = 50
  • C = 100
  • D = 500
  • M = 1000

Caesar Cipher

A Caesar cipher (or a shift cipher), is a cipher where the meanings of the letters are shifted by some set amount.

The most common version is the ROT13 cipher which shifts the values of the letters by 13 places.

Converts any inputted text into a Caesar's Cipher
Green = Terra
Converts any inputted Caesar Cipher back to Text
Terra = Green

Cash Register

This is a cash register function that takes in inputs of (1)Price and then (2)Payment according to the amounts of currency within the cash register and gives you the exact change.


The (1)Price input must be less than the (2)Payment input.


Returns {status: "INSUFFICIENT_FUNDS", change: []} if the cash-in-drawer is less than the change due, or if it can't return the exact change.


Returns {status: "CLOSED", change: ["Your Change"]} if all values are entered and the only currency left is exactly the change amount.


Returns {status: "OPEN", change: ["Your Change"]}, with the change in coins and bills, and it is sorted from highest to lowest order.
